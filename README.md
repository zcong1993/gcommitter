# gcommitter

> Easy way of git commit and push

just `git add -A && git commit -m "msg" && git push` in one command

## Usage:
> download the package, put in any `$PATH` folder.
```bash
# in your work folder
$ gcomitter [flag] [commit msg]
# example
$ gcomitter init
# commit and push
$ gcomitter -p init
```
